#Project 1 - WeatherVue
WeatherVue is a modern weather application crafted to deliver real-time weather updates and forecasts through a user-friendly interface.

## Features
Real-time updates on current weather conditions
Hourly and daily forecasts for planning ahead
Responsive design optimized for both mobile and desktop devices
Getting Started
To run WeatherVue on your local machine:

git clone : https://github.com/poojasawant973/CBTCIP.git
cd Task1
cd weather-application
npm install
npm run dev
Open http://localhost:3000 in your web browser.

## Technologies Used
- Built with React.js for building dynamic user interfaces
- Utilized Axios for handling API requests efficiently
- Styled-components for modular and styled components
- APIs
- WeatherVue fetches weather data using the OpenWeatherMap API. Check out the API Documentation  for more details.
